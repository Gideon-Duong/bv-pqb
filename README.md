# Prisma Query Builder

A TypeScript-based query builder utility for Prisma that helps you build complex queries in a type-safe and intuitive way.

## Installation

```bash
# Using npm
npm install bv-pqb

# Using yarn
yarn add bv-pqb

# Using pnpm
pnpm add bv-pqb
```
